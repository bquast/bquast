# Bastiaan Quast

## Current projects
- [autoresearch-mesa](https://github.com/bquast/autoresearch-mesa) - an adaptation of Karpathy's autoresearch, in turn an OpenClaw / MoldBook style AI agent to explore hyperparameters. Uses the python ABM framework [mesa](https://github.com/mesa).
- [autoresearchABM](https://github.com/bquast/autoresearchABM) - an adaptation of Karpathy's autoresearch, in turn an OpenClaw / MoldBook style AI agent to explore hyperparameters. This framework is adapted to task the AI agent to fine-tune the parameters of an ABM model, such as as the epidemiological SIR recovery model. Custom-built ABM framework that runs headless, logs everything.
